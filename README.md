# Student-Attention-Analysis
This project develops a post-class classroom attention monitoring system that analyzes recorded classroom sessions to measure student visual attention without real-time surveillance. The system detects faces using MediaPipe Face Mesh, estimates head orientation (yaw, pitch, roll) via OpenCV solvePnP, and measures eye alertness through Eye Aspect Ratio (EAR). Each detected student's attention is classified into three levels — Attentive, Inattentive, and Away — using a transparent rule-based system. Outputs include annotated videos, structured CSV exports, and an interactive web dashboard for educator review.
